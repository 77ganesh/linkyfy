Linkyfy : 
--------
Experimental Alpha Version 0.1

A Connectify like GUI utility for creating WiFi hotspots and sharing internet 
connection in GNU/Linux

Date	: 28/03/2015

Dependencies:
-------------
* python2
* pygobject (GTK+ python binding)
* NetworkManager (nmcli)
* hostapd
* isc-dhcp-server
